## Critical Architecture Understanding

### What the Hint System Actually Is

1. **HintVerbosity (Beginner/Intermediate/Expert)** controls DISPLAY DETAIL ONLY for human players
2. **ALL human players should get MCTS (best AI) recommendations**
3. **The 3-box "Hint Comparison" UI is a DIAGNOSTIC TOOL** for the developer to compare what BasicBot/Greedy/MonteCarlo each recommend
4. **Random/BasicBot/Greedy/MonteCarlo are bot difficulty levels** - NOT hint verbosity levels

### Backend Status: COMPLETE

- Charleston tile selection logic EXISTS at `crates/mahjong_core/src/bot/basic.rs:76-105` (`choose_charleston_tiles`)
- MahjongAI trait defines `select_charleston_tiles` at `crates/mahjong_ai/src/trait.rs:67-73`
- All AIs implement this (BasicBotAI wraps `choose_charleston_tiles`)
- Scoring heuristics protect jokers (+100 score, never passed) and prefer passing singletons
