# American Mahjong Asset Generation Guide (DALL-E 3)

This guide is designed to help you generate a consistent set of isometric, 3D-looking American Mahjong tiles using an AI image generator like DALL-E 3 (via ChatGPT Plus or Bing Image Creator).

## Instructions for Use

1. **Copy the Master Style Prompt:** This text block defines the lighting, angle, material, and background.
2. **Choose a Specific Tile Prompt:** Select the tile you want to generate.
3. **Combine and Run:** Paste the Master Style prompt _first_, immediately followed by the Specific Tile prompt, into DALL-E.
4. **Post-Processing:** You will need to use an image editor (Photoshop, GIMP, Photopea) to remove the background shadow and felt, resulting in a transparent PNG sprite.

---

## The Master Style Prompt

**IMPORTANT:** Copy and paste this exact block _before_ every specific tile prompt below.

> An isometric photograph of a single American Mahjong tile, rendered in a highly polished, realistic style. The tile is a thick rectangular block with rounded corners and beveled edges. It is made from a uniform, creamy white bone material on both the front face and the solid back and sides. The tile casts a soft, realistic shadow onto a neutral grey felt background. The carved designs on the face are deep, clean, and appear hand-painted with enamel.
