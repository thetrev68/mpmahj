# User Stories

This directory contains user stories that define the acceptance criteria for each feature. Each user story follows the format:

```text
As a [user type]
I want [goal]
So that [benefit]
```

## File Naming Convention

`US-XXX-[feature-name].md`

Where:

- `XXX` = Sequential number (001, 002, etc.)
- `[feature-name]` = Kebab-case description

See `../../../scratchpad/user-story-template.md` for the complete template.
